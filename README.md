# 🎧 Blogger Audio Player

A lightweight, customizable **audio player widget for Blogger** using HTML, CSS, and JavaScript.  
This project allows you to host assets via GitHub + jsDelivr CDN and embed a fully functional music player inside Blogger websites.

---

## 🚀 Features

- 🎵 Playlist support via `data-tracks`
- ⏯ Play / Pause / Next / Previous controls
- 📊 Audio progress bar
- 🎨 Animated visualizer
- 📱 Fully responsive design
- ⚡ CDN-based CSS & JS (fast loading)
- 🔧 Easy MP3 replacement without editing code

---

## 📦 Guide

### 1. Add to Blogger

Inside `<head>`:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/PashtunObserver/blogger-audio-player@main/style.css"/>

Before `</body>`:

<script src="https://cdn.jsdelivr.net/gh/PashtunObserver/blogger-audio-player@main/script.js"></script>

---

### 5. Add Widget

Paste the HTML widget into Blogger Layout → HTML/JavaScript gadget.

---

## 🎶 Update Songs

Edit only:

data-tracks='[
{
  "title":"Song Name",
  "artist":"Artist Name",
  "src":"https://yourlink/song.mp3",
  "duration":"3:40"
}
]'

---

## ⚠️ Requirements

- Direct .mp3 links only  
- HTTPS required  
- Public access required
